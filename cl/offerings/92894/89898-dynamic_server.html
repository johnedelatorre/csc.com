Dynamic Server

Dynamic Server permite a las empresas proporcionar recursos informáticos para ahorrar costos

CSC Dynamic Server es un servicio entregado bajo la modalidad de Infraestructura como Servicio (IaaS) para empresas.

Admite varias plataformas de servidores midrange, autoaprovisionamiento y la modificación de recursos informáticos a través de un catálogo de servicios basado en la red. El servicio se entrega mediante una arquitectura de referencia independiente que incluye recursos de servidores, redes, almacenamiento, y viene integrado con un marco de gestión de servicios ITIL (Information Technology Infrastructure Library).

CSC Dynamic Server es un servicio con un modelo de precios de pago por consumo que ayuda a empresas de cualquier industria a evitar costos de TI asociados con capacidad no utilizada.

Beneficios comerciales y técnicos

CSC Dynamic Server aumenta la flexibilidad en la provisión de servicios de TI. Con los servicios tradicionales de TI, los tiempos de aprovisionamiento suelen ser de 30 a 90 días; no obstante, CSC Dynamic Server entrega capacidad de TI en una fracción de ese tiempo con un promedio de sólo 48 horas. Mediante la capacidad de congelar y descongelar del servicio, el negocio puede administrar mejor los presupuestos de TI.

CSC Dynamic Server es un chasis blade con la cantidad adecuada de blades para admitir la cantidad requerida de instancias virtuales para un sitio determinado. Los servidores blade ejecutan VMware para virtualización y están conectados con la infraestructura de almacenamiento y respaldo compartida del Data Center. Dynamic Server también incluye un motor de aprovisionamiento que proporciona un front end web que permite capacidades de aprovisionamiento dinámicas.

Junto a lo anterior, Dynamic Server implementa herramientas de gestión de servicios empresariales que proporcionan monitoreo y generación de informes en tiempo real y predictivas para ayudar a identificar problemas antes de que surjan. Con Dynamic Server, las empresas pueden responder cómodamente a requerimientos de seguridad y cumplimiento, operando programas de manera más eficiente que con los ambientes de tecnología y hospedaje tradicionales.


Diferenciación de CSC

CSC Dynamic Server:

  • Proporciona un servicio de nube segura y privada, diseñado para clientes empresariales.
  • Entrega capacidad de autoservicio y computo en demanda aprovechando un catálogo de servicios personalizable.
  • Entrega libertad y flexibilidad a los desarrolladores al permitirles usar plataformas y herramientas de su elección.
  • Está configurado para permitir que las empresas determinen su propio nivel de seguridad.
  • Proporciona gestión del ciclo de vida completamente automatizada la cual, incluye provisión, modificación, congelación, descongelación, archivo y baja de servicio.
  • Ofrece modelos de disponibilidad para garantizar que sus sistemas de negocios estén listos cuando los necesite.

Retorno de la inversión

La virtualización y la consolidación ayudan a administrar el costo creciente de TI, al reducir el espacio físico que ocupa el servidor. Las empresas pueden evitar la inversión en activos de capital con CSC Dynamic Server, porque les permite consumir TI como un servicio. Así mismo, al no requerir reingeniería, CSC Dynamic Server brinda una transición breve y un costo bajo de entrada.

CSC Dynamic Server proporciona la visibilidad y la posibilidad de gestión automatizada de la capacidad del servidor. El servicio permite a las gerencias de TI administrar mejor la demanda no prevista y responder rápidamente a climas de negocio cambiantes, así como implementar más rápidamente nuevos productos en el mercado. Los clientes pueden reducir y contener los costos asociados con la capacidad no utilizada.
